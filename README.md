## Getting Started

### Prerequisites

- Java (update as per your stack)
- Maven (update as per your stack)

### Installation

```bash
# Clone the repository
git clone <repo-url>
cd storemanagementapp

# Install dependencies
# install macport
# google it macport download for mac

# install git 

sudo port install git

sudo port install maven3

# clean dependencies
mvn clean install

### Running the App

mvn spring-boot:run 
```



```bash
npm start
# or
python app.py
# or
mvn spring-boot:run
```

### Running Tests

```bash
npm test
# or
pytest
# or
mvn test
```
